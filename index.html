PJ Ramen Shop

Oct 2021

PJ Ramen Shop is a ramen-customization simulator that I created in a team of two in the span of five days. We used HTML, CSS, and JavaScript to create the web app, as well as Adobe Illustrator and Autodesk Sketchbook for the graphics. This project won 2nd place in ICSSC's WebJam, which you can read about here!

Columns

Nov 2021

This project is a recreation of the game Columns. The game mechanics and the UI were made with Python and the Pygame library.

Dogs vs Cats

Jan 2021 - Mar 2021

Dogs vs Cats is a tower defense game made using C++ and SFML. I collaborated with four teammates for this project, learning how to use GitHub in the process. I primarily focused on designing the UI, programming the main menu, and creating the sprites and their animations while implementing them into the game.

Grow

Feb 2022

Grow is a work in progress. It is a minigame that will allow you to take care of a plant and watch it grow into something unexpected. I am currently developing it in my free time, using React (which I am learning). Although it is still unfinished, feel free to take a look!
